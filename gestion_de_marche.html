Gestion des Marchés

Contexte du projet

Une chargée de commerce, en charge de la relation client et du suivi des demandes commerciales, a exprimé le besoin d’un outil pour mieux organiser les marchés en cours et à venir. Jusqu’à présent, les informations étaient réparties entre différents outils, ce qui compliquait la centralisation, la réactivité et la visibilité globale sur les marchés traités.

Pour répondre à cette problématique, un système de gestion des marchés a été conçu. Il est composé de deux modules complémentaires :

  • Un module de gestion pour organiser les marchés (création, modification, suppression)
  • Un module de supervision cartographique pour visualiser les marchés géographiquement sur une carte de France

1. Vue d’ensemble de l’application

L'application repose sur deux modules principaux :

  • Gestion des Marchés : consultation, création, modification et suppression
  • Cartographie des Marchés : visualisation géographique interactive

2. Module 1 : Gestion des Marchés

2.1 Accès à la liste des marchés

  • Accès via le menu "Marchés"
  • Affichage des marchés sous forme de tableau avec colonnes : Ville, Client, Intitulé du marché", Produit(s), Chargé(e) de projet et Commercial

2.2 Ajouter un marché

  • Bouton "Ajouter un marché" disponible au-dessus du tableau
  • Formulaire à remplir avec : nom, adresse, responsable, statut, logo, etc.
  • Coordonnées GPS calculées automatiquement à partir de l’adresse

2.3 Modifier ou supprimer un marché

  • Un clic sur un marché ouvre la fiche pré-remplie
  • Possibilité de modifier les informations ou de supprimer le marché

3. Module 2 : Cartographie des Marchés

3.1 Affichage de la carte

  • Carte de France interactive avec marqueurs pour chaque marché
  • Coordonnées GPS utilisées pour positionner les marchés

3.2 Interaction avec les marqueurs

  • Affichage d’une infobulle au clic sur un marqueur
  • Informations : nom, description, adresse, responsable, statut et logo

3.3 Filtres

  • Filtres par responsable, statut ou type de produit pour affiner l’affichage

4. Technologies et API

  • GET /project : récupérer la liste des marchés
  • POST /project : créer un marché
  • PUT /project/{id} : modifier un marché
  • DELETE /project/{id} : supprimer un marché
  • GET /manager : récupérer les responsables
  • GET /product : récupérer les produits
  • GET /sale : récupérer les commerciaux

5. Conseils d’utilisation

  • Connexion Internet requise
  • Vérifier les champs obligatoires avant validation
  • Confirmer les coordonnées GPS pour une bonne géolocalisation

6. Évolutions possibles

  • Ajout de pièces jointes (contrats, photos...)
  • Export des données (PDF, Excel)
  • Indicateurs de suivi et de performance
  • Notifications ou rappels automatisés
