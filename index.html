C'est quoi le BTS SIO SLAM ?

Le BTS SIO SLAM (parcours Informatiques aux Organisations, Solutions Logicielles et Applications Métiers) est un diplôme en 2 ans après le bac. Cette formation est idéale pour ceux qui souhaitent se former en informatique tout en acquérant une expérience professionnelle grâce à l’alternance.

Métiers visés

Les métiers visés par cette formation sont variés, tels que :

  • Développeur web
  • Développeur d’applications mobiles
  • Analyste d’applications
  • Technicien de maintenance en informatique

Objectifs de la formation

Le BTS SIO SLAM vise à former des professionnels capables de :

  • Gérer l’ensemble informatique d’une entreprise
  • Développer et maintenir des solutions logicielles
  • Former le personnel aux nouvelles applications

Contenu de la formation

La formation comprend des cours dans les domaines suivants :

  • Conception et développement d’applications
  • Maintenance et gestion de systèmes informatiques
  • Cybersécurité des parcours informatiques
  • Développement web et objet
  • Analyse de données

Débouchés

Après cette formation, plusieurs opportunités s'offrent à vous, notamment :

  • Travailler dans des entreprises de parcours du numérique
  • Rejoindre des éditeurs de logiciels
  • Intégrer les parcours informatiques d’entreprises
  • Poursuivre ses études en licence professionnelle ou en école d’ingénieurs

Veille technologique

Voici les thématiques que je surveille régulièrement pour rester à jour dans le domaine de l'informatique.

Intelligence Artificielle & Machine Learning
Définition

L'IA simule des comportements humains. Le Machine Learning (ML) entraîne un modèle à partir de données pour faire des prédictions sans codage explicite.

Exemples concrets

Reconnaissance de chiffres manuscrits, prévision météo, recommandations YouTube, détection de spams.

Étapes d'un projet ML

Collecte de données, prétraitement, choix de l'algorithme (régression, arbres, réseaux), évaluation, déploiement.

Pourquoi ça m'intéresse

Applications en cybersécurité, objets connectés, automatisation, analyse de données. Vision élargie de l'informatique.

Frameworks JavaScript
Vite

Outil de build ultra-rapide avec rechargement à chaud. Gain de temps énorme en dev front-end.

Tailwind CSS

Framework utility-first, création rapide d’interfaces modernes, responsive, sans écrire de CSS personnalisé.

Bootstrap

Système de grille fluide, composants prêts à l’emploi, carrousel, modales. Compatible tous navigateurs.

Utilisation de Git
Utilisation en entreprise

Chaque développeur travaille sur sa propre branche pour éviter les conflits. Les projets sont hébergés sur un serveur Git (souvent via SSH) avec des commandes telles que git clone, git pull, git push, git merge, git rebase.

Exemple concret

Travail en feature branch avec envoi via merge request après revue du code, garantissant une intégration propre à la branche main.

Utilisation de GitHub

Partage de projets personnels, collaboration open source, veille sur projets populaires (Tailwind, Laravel, Astro...).

Avantages observés

Historique clair, travail collaboratif fluide, sécurité via sauvegarde distante, responsabilisation de chaque contributeur.

Pourquoi c’est essentiel pour moi

Git m’a appris à coder avec rigueur, à rédiger des commits explicites et à maintenir un historique propre.
